# Llm Collector

A Mintlify docs site for tracking model/vendor/platform intelligence.

## What it collects
- New model launches
- Deprecations and shutdowns
- Pricing changes
- API and compatibility updates
- Aggregator/platform changes

## How to use
1. Review `guide/sources` and add a source in `[sources/index.yml](https://github.com/albertus20001-design/llm-collector/blob/main/sources/index.yml)`
2. Record collection cadence in `guide/schedule`
3. Publish findings in `guide/output`
